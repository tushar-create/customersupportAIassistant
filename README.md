# Customer Support AI System

## Tech Stack
- .NET Core 8 Web API
- React + TypeScript
- OpenAI / Groq LLM
- RAG-based Knowledge Base

## Features
- AI-generated support drafts
- Human-in-the-loop approval workflow
- FAQ + policy-based grounding
- Audit logging using file storage

## Run Backend
dotnet run

## Run Frontend
npm install
npm start
